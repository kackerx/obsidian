# error
	
- Sentinel Error: 哨兵

> 中间件, 第三方库用的多, 业务开发用的少, 要少而精, 开发者觉得用户需要做特殊处理的错误, 才标记出来, NoRows

- Error types

> error要遵循一定规范, 比如error要有error code, 可以自定义错误类型, 但也要避免暴露出去, 用户永远只需要使用error接口

- Opaque errors
