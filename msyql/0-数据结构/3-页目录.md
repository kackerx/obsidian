- 每组最后一条记录的头信息中的n_owned表示该组几条记录
- 每组最后一条记录的地址偏移量存入page directory, 作为槽, slot

## 查找过程
	
- 查找主键为6, 计算中间槽位置 (0 + 4 ) // 2 = 2
- 槽2对应为8 > 6, high = 2, 二分(0 + 2) // 2 = 1, 槽1对应为4 < 6, low = 1
- 确定在槽2中, 槽是相邻的, 我们拿到槽1对应的记录, 下一条就是槽2的最小值, 然后开始遍历找到6
