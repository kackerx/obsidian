# 核心对象
## Node

~~~ad-primary
title: Node

计算节点的抽象, 用来描述计算节点的资源抽象, 健康状态
~~~

## Namespace

~~~ad-grey
title: Namespace

资源隔离的基本单位, 简单理解为文件系统中的目录结构
~~~

## Pod

~~~ad-success
title: Pod

描述应用的实例, 包括镜像地址, 资源需求等, k8s中最核心对象, 打通应用和基础架构的秘密武器
~~~

title: Service
