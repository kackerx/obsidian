## sumup

> ES是最基本语法的脚本语言, 看作是js的标准化规范, js实现了ES的标准, ES6就是ESMAScript2015

- JS是ES基础上的拓展, JS是ES的扩展语言
- 浏览器中: JS = ES + Web Apis (bom + dom)
- Node中: JS = ES + Node Apis (fs + net)

## web

- DOM: 文档 对象 模型, 操作节点树
- BOM: 浏览器 对象 模型, 操作浏览器窗口, 等的能力
